Alerts

Alert messages can be used to notify the user about something special: danger, success, information or warning.

Example:

× Danger! Indicates a dangerous or potentially negative action.

Code:

                
                   <div class ="alert-danger">
                       < span class ="closebtn" onclick="this.parentElement.style.display='none';" >&times;<span>
                        Danger! Indicates a dangerous or potentially negative action.
                    <div>

× Success! Indicates a successful or positive action.

Code:

                    
                       <div class ="alert-success">
                           < span class ="closebtn" onclick="this.parentElement.style.display='none';" >&times;<span>
                            Success! Indicates a successful or positive action.
                        <div>

× Info! Indicates a neutral informative change or action.

Code:

                  
                     <div class ="alert-info">
                         < span class ="closebtn" onclick="this.parentElement.style.display='none';" >&times;<span>
                          Info! Indicates a neutral informative change or action.
                      <div>

× Warning! Indicates a warning that might need attention.

Code:

                
                   <div class ="alert-warning">
                       < span class ="closebtn" onclick="this.parentElement.style.display='none';" >&times;<span>
                        Warning! Indicates a warning that might need attention.
                    <div>

Buttons

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

Example:

Primary Buttons

Code:

              
                  <div>
                      <button class="prim-success">Success</button>
                      <button class="prim-info">Info</button>
                      <button class="prim-warning">Warning</button>
                      <button class="prim-danger">Danger</button>
                      <button class="prim-default">Default</button>
                   </div>

Secondary Buttons

Code:

              
                  <div>
                      <button class="btn-success">Success</button>
                      <button class="btn-info">Info</button>
                      <button class="btn-warning">Warning</button>
                      <button class="btn-danger">Danger</button>
                      <button class="btn-default">Default</button>
                   </div>

Lists

List component is used to display list items. It renders a ul element by default.

Example:

Inline Lists

  • Home
  • Docs
  • About Me

Code:

              
                  <ul class="inline-list">
                      <li class="inline-items"><a href="#" class="inline-link link-active">Home</a></li>
                      <li class="inline-items"><a href="#" class="inline-link">Docs</a></li>
                      <li class="inline-items"><a href="#" class="inline-link">About Me</a></li>
                  </ul>

Stacked Lists

  • About Me
  • Docs
  • Home

Code:

              
                  <div class="stack-list">
                      <ul>
                          <li style="list-style: none;">About Me</li>
                          <li style="list-style: none;">Docs</li>
                          <li style="list-style: none;">Home</li>
                      </ul>
                  </div>
